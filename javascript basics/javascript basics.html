Javascript Basics


Javascript is a cross-platform, onject-oriented scripting language. Javascript is extremely popular for a variety ov reasons. It is a small and lightweight language allowing maximum flexibility for develoopers to take it in a bunch of different directions. Javascript lives inside a host environment (a web browser or Node sserver), it can be connected to the objects of these environments to provide programmatic control over them.


  • Variable Declaration Javascript variables are containers for storing data values-imagine a cup you fill with coffee, the cup holds the coffee, a variable holds a value. All Javascript varianles must be identified with unique names. These unique names are called identities. var x;

  • Variable Assignment Assignment operators assign values to Javascript variables-our cup can now have coffee poured in it, giving our variable a value to hold. The = assignment operator assigns a value to a variable. var x = 10;

  • Data TypesData types are an important concept; to be able to operate on variables you need to know the data type.
